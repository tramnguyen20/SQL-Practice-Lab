# SQL Technical Practice Lab: From Fundamentals to Advanced Database Objects

## Project Overview
Kho lưu trữ này là tập hợp các bài tập thực hành kỹ thuật SQL (Technical Lab), hệ thống hóa toàn bộ kiến thức từ xử lý dữ liệu cơ bản đến các đối tượng nâng cao trong quản trị cơ sở dữ liệu.

## Technical Skill Map

### 1. Data Definition & Integrity (DDL)
*Files: ddl_and_constraints.sql*
- Thiết kế cấu trúc bảng và quản lý ràng buộc dữ liệu (Primary Key, Foreign Key, Check).
- Tư duy xây dựng hệ thống dữ liệu nhất quán, chống sai sót ngay từ bước khởi tạo.

### 2. Data Manipulation & Logic (DML)
*Files: dml_and_transactions.sql, where_between_in.sql, set_operators.sql*
- Thực thi các thao tác cập nhật dữ liệu và kiểm soát giao dịch (Transactions).
- Sử dụng các bộ lọc logic phức tạp để phân loại và truy vấn tệp khách hàng mục tiêu.

### 3. Advanced Aggregation & Join Patterns
*Files: groupby_and_aggregate.sql, joins_groupby_subquery.sql, sql_join_exercises.sql*
- Kỹ thuật gom nhóm dữ liệu (Aggregation) để tính toán các chỉ số tổng hợp.
- Kết hợp dữ liệu đa tầng (Multi-table Joins) và truy vấn lồng (Subqueries) để giải quyết các yêu cầu báo cáo chéo.

### 4. Database Objects & Optimization
*Files: views_and_joins.sql, views_procedures_phieuxuathu.sql, subquery_practice.sql*
- **Views**: Đóng gói các logic truy vấn phức tạp thành bảng ảo, tối ưu hóa việc tái sử dụng code.
- **Stored Procedures**: Tự động hóa các luồng xử lý dữ liệu.

## Learning Path Representation
Toàn bộ mã nguồn minh chứng cho khả năng làm chủ SQL qua các giai đoạn:
1. **Foundation**: Làm quen với cú pháp và cấu trúc dữ liệu.
2. **Analysis**: Sử dụng SQL như một công cụ phân tích (Joins, Aggregation).
3. **Engineering**: Xây dựng các đối tượng tối ưu (Procedures, Views) để xử lý dữ liệu tự động.
